# IoT-Firstep
一个物联网开发的入门教程。涉及单片机、上位机、移动应用、服务器后台开发的知识。以及BLE、WiFi使用实例。
## 状态
努力编写中。
## 说明
本文的开发环境除了iOS的开发，都是在Windows平台进行的。当然，如果不是进行单片机的开发或者是.Net平台下的的开发的话，推荐最好还是不要选择Windows系统。
